Species List Builder Demo

Type your search terms into the input box. Click on individual taxa in the drop-down list to add them to the right-hand side selection list. Clicking on an item in the selection list will remove it from the list. Pressing the submit button will return the selected items.

By default, selected items are sorted alphabetically but this can be overridden if desired. Also, it is possible to abbreviate the selected list to display only scientific names. The 'Submit' button is displayed only if a callback function is provided as a prop.
